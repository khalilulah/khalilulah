## 🚀 Featured Projects

### [Job Queue & Background Worker System](https://github.com/khalilulah/jobQueue)
Redis-backed job queue built from first principles — atomic job reservation, retry scheduling with exponential backoff, crash recovery, visibility timeouts, graceful shutdown, and dead-letter handling.
`Node.js` `TypeScript` `Redis (Upstash)` `ioredis` `Jest`

### [Multi-Tenant SaaS API with RBAC](https://github.com/khalilulah/saas-rbac-api)
Multi-tenant API using shared-schema isolation and Postgres Row-Level Security, with dual authentication (JWT + API keys) and a `SECURITY DEFINER` function for atomic organization creation.
`Node.js` `TypeScript` `Express` `PostgreSQL` `Supabase` `Jest/Supertest`

### [RealWorld Conduit API](https://github.com/khalilulah/conduit-api)
20-endpoint REST API (articles, comments, favorites, follows, tags) built on raw SQL — no ORM — plus rate limiting and request-logging middleware.
`Node.js` `Express` `PostgreSQL` `Jest/Supertest`

### [Muraqib — Quran Recitation Accountability App](https://github.com/khalilulah/muraqib_server#)
Full-stack app with a Node/Express backend handling OAuth2 (PKCE) integration, push notification delivery, audio upload pipelines, and recitation similarity scoring — plus a native Android Focus Mode via Kotlin AccessibilityService.
`React Native (Expo)` `Node.js/Express/TypeScript` `PostgreSQL` `Supabase`

### Nigerian Tax Chatbot — Scribble Green
